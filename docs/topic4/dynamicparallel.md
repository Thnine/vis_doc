# 面向二部图结构演化的动态弦图

本组件将动态网络依时序横向平行排列。在每个时间片内，则依据子二部图的节点个数数量从上到下垂直排列；对于每一个子二部图，使

用双聚类算法对节点排序。该算法兼顾同一侧以及两侧节点间的相似性，使得连接关系相近的节点排在两侧的相近位置。排序后，将左侧

节点置于布局上侧，右侧节点置于布局下侧，并使用蓝色曲线表示左右节点的连接关系。

## 输入数据

- 描述：点数据与边数据，要求根据边数据得到的连通分量为二部图。
- 类型：`object`，满足`json`格式

数据中的`graph_nodes`为存储点数据的对象数组，`graph_edges`为存储边数据的对象数组。

点数据要含有唯一标识的`id`，为`Number`类型，`ip`，为`String`类型；边数据要包含起点`id`、终点`id`，以及该边所处时间段。

数据的样例请详见章节[使用示例](#使用示例)

## 如何使用

### 启动

```javascript
//预处理数据，data为传入数据
this.handleMockData("data");
```

### 交互

在该组件上，可以实现以下交互：

- **缩放与拖曳**：通过鼠标滚轮进行放大、缩小，按住鼠标进行拖动以观察节点的详细信息。
- **悬停查看节点信息**：将组件放大后，可将鼠标悬停于想观察的节点上，会出现浮框显示该节点的属性，移开鼠标浮框消失
- **高亮输入点**：点击高亮输入点测试按钮，可以将外部接口`highlightNodes`中输入的点集合高亮
- **高亮时间线**：点击高亮时间线接口按钮，可以将外部接口`highlightTimeLine`中输入的点所对应的时间连线高亮
- **导出节点**:点击节点，该节点高亮，可以通过外部接口`getNodeInfo`将该节点的信息导出，

## 外部接口

### 发送

#### 点击节点

- 接口名称：`getNodeInfo`

- 交互描述：鼠标点击节点，见该节点信息导出

- 数据格式：`object`，表示导出的节点对象

### 接收

**高亮节点**

- 接口名称：`highlightNodes`
- 交互描述：鼠标点击高亮输入点测试按钮，将节点高亮
- 数据格式：list,表示被高亮节点的`ip`列表

**高亮时间线**

- 接口名称：`highlightTimeLine`
- 交互描述：点击高亮时间线接口按钮，可以将输入的点所对应的时间片连线高亮
- 数据格式：`String`,表示该节点`ip`
